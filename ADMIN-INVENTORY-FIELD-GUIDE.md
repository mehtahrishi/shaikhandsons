# 📋 Admin Inventory - Complete Field Guide

## How to Add a New Vehicle (Add New Asset)

This guide shows **exactly what to select or type** for every field in the inventory form.

---

## 🔧 SECTION 1: BASIC INFORMATION

### 1️⃣ Manufacturer (Brand) - **SELECT from Dropdown**
**What to do:** Click the field and choose from list
**Options available:**
- Dynamo
- Hero
- Bajaj
- TVS
- Ather
- *(or any brand already created in Brand Management)*

**Example:** Select "Dynamo"

---

### 2️⃣ Model Name - **TYPE Text**
**What to do:** Type the model/variant name
**Format:** Any name (1-2 words)
**Examples:**
- `Lima`
- `Apex Pro`
- `Thunder Elite`
- `Swift Standard`

---

### 3️⃣ Model Number/Code - **TYPE Text**
**What to do:** Type the internal model code
**Format:** Alphanumeric code (letters + numbers)
**Examples:**
- `LIMA-STD`
- `APEX-2026`
- `THU-PRO-V2`
- `SWIFT-BASE`

---

### 4️⃣ Vehicle Category - **SELECT from Dropdown**
**What to do:** Click and choose category type
**Options available:**
- `Scooter` ← Most available
- `Bike`
- `Loader`

**Example:** Select "Scooter"

---

### 5️⃣ Ex-Showroom Price (₹) - **TYPE Number**
**What to do:** Enter price in Indian Rupees
**Format:** Numbers only (no commas, no ₹ symbol)
**Examples:**
- `70000`
- `85000`
- `45000`
- `120000`

---

### 6️⃣ Year - **TYPE Number**
**What to do:** Enter manufacturing/launch year
**Format:** 4-digit year
**Examples:**
- `2024`
- `2025`
- `2026`

---

### 7️⃣ Variant/Trim - **TYPE Text (Optional)**
**What to do:** Enter trim level (can be left blank)
**Format:** Text (optional)
**Examples:**
- `Grand Touring`
- `Standard`
- `Premium`
- `Base Model`
- *(leave empty if no trim)*

---

### 8️⃣ Short Description - **TYPE Text**
**What to do:** Write a 1-2 sentence hook for listing page
**Format:** Text (50-100 characters)
**Examples:**
- `Affordable low-speed electric scooter with smart features`
- `Premium performance scooter with extended range`
- `Budget-friendly commuter perfect for city travel`

---

## ⚡ SECTION 2: TECHNICAL SPECIFICATIONS

### 9️⃣ Top Speed - **TYPE Text**
**What to do:** Enter maximum speed
**Format:** Number with unit (e.g., "45 km/h")
**Examples:**
- `25 km/h`
- `45 km/h`
- `60 km/h`
- `80 km/h`

---

### 🔟 Certified Range (ARAI) - **TYPE Text**
**What to do:** Official certified range per charge
**Format:** Number with unit (e.g., "120 km/charge")
**Examples:**
- `120 km/charge`
- `180 km/charge`
- `250 km/charge`

---

### 1️⃣1️⃣ Real-World Range - **TYPE Text**
**What to do:** Practical range in real conditions
**Format:** Range format (e.g., "90–100 km")
**Examples:**
- `90–100 km`
- `120–150 km`
- `150–180 km`

---

### 1️⃣2️⃣ Riding Modes - **SELECT Checkboxes** (Multiple selection)
**What to do:** Check all that apply
**Options available:**
- ✅ `Eco` - Most efficient, lower power
- ✅ `City` - Balanced for city riding
- ✅ `Sport` - Higher performance
- ✅ `Reverse` - Backward movement

**Example:** Check "Eco" and "City"

---

### 1️⃣3️⃣ Climbing Degree - **TYPE Text**
**What to do:** Maximum incline angle capability
**Format:** Angle in degrees (e.g., "15 Degrees")
**Examples:**
- `7–10 degrees`
- `15 degrees`
- `20 degrees`

---

### 1️⃣4️⃣ Load Capacity - **TYPE Text**
**What to do:** Maximum weight it can carry
**Format:** Weight in kg (e.g., "180 kg")
**Examples:**
- `180 kg`
- `200 kg`
- `250 kg`

---

## 🔋 SECTION 3: BATTERY & CHARGING

## ⚠️ IMPORTANT NOTES

1. **Required Fields** (must fill):
   - Brand ← Must select
   - Model ← Must type
   - Price ← Must enter number
   - Year ← Must enter number

2. **Optional Fields** (can leave blank):
   - Trim, Description, Images, Colors, Features (all can be empty)

3. **Colors Format**:
   - Type like this: `Red, Blue, Silver`
   - Click outside field to save
   - Each color separated by comma + space

4. **Images**:
   - If you upload images → They're used
   - If you leave empty → Beautiful placeholders auto-generated
   - Both work fine!

5. **Dropdowns**:
   - Click to open list
   - Click option to select
   - Can't type, only select from list

6. **Text Fields**:
   - Type freely
   - Can use spaces and special characters (except in numbers)
   - No length limit (but be reasonable, 50-200 chars best)

---

## 🆘 COMMON MISTAKES TO AVOID

❌ **Don't do this:**
- Leave Brand empty (required!)
- Type number with commas: `70,000` (type as `70000`)
- Add ₹ symbol in price field (just number)
- Forget to click outside colors field (click elsewhere after typing)

✅ **Do this instead:**
- Always select a Brand first
- Type prices as: `70000`
- Colors: Type `Red, Blue` then click tab or another field
- Review before clicking Save
