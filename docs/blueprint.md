# **App Name**: Veridian Noir

## Core Features:

- Electronic Vehicle Showroom: Display a curated selection of electronic vehicles with high-quality images, specifications, and key features, allowing for vehicle-centric grid layouts.
- Vehicle Detail Pages: Provide dedicated pages for each electronic vehicle model, showcasing detailed specifications, performance metrics, additional imagery, pricing, and options for configuration or inquiry.
- Vehicle Inquiry & Reservation System: Allow users to express interest in specific vehicles, request quotes, schedule test drives, or place reservations through a structured inquiry process.
- User Authentication & Profiles: Implement basic user registration and login, allowing users to manage their vehicle inquiries, reservations, and view their interaction history.
- Dynamic Dual-Theme Toggle: Allow users to switch between light and dark modes, with the specified dark mode set as the primary default experience.
- AI Vehicle Description & Configuration Tool: An administrative tool to generate engaging and editorial-style vehicle descriptions based on input specifications and features, and to assist in suggesting optimal vehicle configurations.

## Style Guidelines:

- Primary UI Accent: A vibrant teal (#00BCD4) to signify interactive elements and emphasize high-fashion appeal in dark mode, fitting a luxury EV brand with a modern edge.
- Background Subtlety: A very deep black (#1A1A1A) to add depth while adhering to a dark primary scheme, enhancing the premium feel.
- Accent Color: A muted, sophisticated grey (#B0BEC5) provides a gentle contrast, ideal for secondary highlights and iconography, maintaining sophistication without competing with the teal.
- Dark Mode Main Foreground: Off-white/Near-white (#fafafa) for primary text and foreground elements, ensuring high readability in the primary dark theme.
- Dark Mode Main Background: Deep charcoal/Black (#0a0a0a) for the default page background, reflecting the modern, high-contrast aesthetic suitable for luxury EVs.
- Destructive Actions: Vibrant Red (#ef4444) to clearly indicate errors or delete actions, ensuring functionality is unambiguous and stands out.
- Muted Elements: Subtle grays for secondary text and borders, maintaining a clean hierarchy within the monochromatic base without distracting from vehicle visuals.
- Headline Font: 'Playfair Display' (serif, weights 700 to 900) for a 'Vogue-style' editorial and luxurious feel for brand identity, page titles, and hero sections, complementing the high-end vehicle aesthetic.
- Body Font: 'Poppins' (sans-serif, weights 300 to 700) for excellent readability in navigation, vehicle descriptions, buttons, and UI controls, providing a modern tech-forward look.
- Element Softness: All interactive elements and cards feature a consistent 0.5rem (8px) border radius for a modern yet slightly softened edge, appropriate for refined design.
- Persistent Header: A fixed 'SiteHeader' with essential navigation links will be present across all pages, ensuring consistent access to key sections like the vehicle showroom and inquiry forms.
- Visual Storytelling Focus: Layouts prioritize large hero banners and vehicle-centric grids, placing heavy emphasis on high-quality imagery to showcase electronic vehicles effectively.
- Global Notifications: Implement 'Global Toaster' notifications for site-wide user feedback and alerts, such as successful inquiries or reservation updates.
- Framer Motion Integration: Extensive use of Framer Motion for smooth page transitions, loading states, and dynamic quotes, enhancing the premium user experience for a luxury EV brand.
- 3D Interactive Elements: Custom 'flip' animations utilizing 3D transforms will be applied to interactive cards, such as vehicle model cards, adding a premium and engaging feel.
- Dynamic Micro-interactions: Include an 'AnnouncementBar', 'ChatWidget', and 'Typewriter' brand effects to create an 'alive' and engaging user experience, reflecting innovation.
- Introductory Custom Loader: A 4-second introductory custom loader animation will build brand anticipation before the main content loads, setting a high-quality tone.
